# Develop

<img src="./public/demo.png">

```bash
# 获取远程仓库代码
git clone https://github.com/Wscats/piano
# 进入目录
cd piano
# 安装依赖
npm install
# 启动项目
npm start
# 在浏览器访问 http://localhost:3000
```

# Release

发布属于自己的演奏版本
```bash
npm run build
```
